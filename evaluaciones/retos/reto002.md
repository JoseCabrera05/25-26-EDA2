# Reto 002

Diseñe un algoritmo que mediante recursividad resuelva el siguiente acertijo:

```
   S E N D
 + M O R E
 ---------
 M O N E Y
```

Una vez que lo haya deducido, pruébelo/ajústelo para resolver:

```
   F O R T Y
       T E N
 +     T E N
 -----------
   S I X T Y
```

o

```
  O D D
+ O D D
--------
 E V E N 
``` 

## Importante

Siga el planteamiento enviado como propuesta
